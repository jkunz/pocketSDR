About This App

This app uses the Adobe Analytics Admin API to store variable settings on your phone for easy reference. Note: this app only observes your current settings- it won't be useful if you want to change or copy settings.

For a shortcut to make it easier to transfer credentials to this app, visit digitalDataTactics.com/pocketSDR/ (ideally, on a non-mobile browser).

PocketSDR was created as an amateur development project by Jenn Kunz of digitalDataTactics.com

I had three main goals:

  • Make it easier for folks to look up a variable's purpose and settings, without going to the Admin Console or documentation.

  • Give me an example to reference to show how to implement DTM in an Angular framework.

  • Give me some hands-on experience implementing Lifecycle Metrics in a mobile app.

This iteration was created using the Ionic 3 framework. Analytics tracking was implemented using both the Adobe Analytics SDK and Adobe Dynamic Tag Manager. This app is open-source. Feel free to see if you can improve on it.
